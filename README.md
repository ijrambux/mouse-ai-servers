# MOUSE AI IPTV Merge — Web App

ملف المشروع يحتوي على:
- index.html — واجهة الويب (PWA).
- manifest.json — تعريف التطبيق لتثبيت PWA.
- sw.js — خدمة التخزين المؤقت (Service Worker).

تشغيل محلي سريع:
1. ضع الملفات في مجلّد واحد.
2. شغّل خادم HTTP بسيط من داخل المجلّد:
   - بايثون 3: `python -m http.server 8000`
   - افتح المتصفح: `http://localhost:8000`
3. الآن تستطيع استيراد السيرفرات، إنشاء M3U وتحميله أو نسخه.

ملاحظات تقنية مهمة:
- التطبيق يعمل كصفحة من جهة العميل (client-side). لا يقوم برفع بياناتك لأي خادم خارجي.
- عند محاولة تحميل ملفات M3U مباشرة من مواقع خارجية قد تواجه قيود CORS. الحلول:
  - استيراد الملف بالمحتوى (نسخ/لصق) داخل مربع الاستيراد.
  - استخدام proxy موثوق (مثلاً corsproxy.io أو proxy خاص بك) — ضع في اعتبارك مخاطر الأمان.
- Service Worker يقوم بتخزين واجهة التطبيق محلياً لتحميل أسرع وإمكانية العمل offline لواجهة التطبيق. لكنه لا يتجاوز سياسات CORS عند جلب محتوى من مصادر خارجية.

اقتراحات مستقبلية:
- إضافة فحص حالة (health-check) لكل سيرفر (تستدعي GET على رابط test) وتحديث الحالة (working/failed).
- واجهة إدارة متقدمة لتجميع السيرفرات حسب البلد/المزود، وتصفية أو حذف جماعي.
- دعم استيراد M3U كبير تدريجياً (streaming parser) مع virtualized list لواجهات سريعة عند آلاف القنوات.

إذا تريد، أجهّز لك:
- نسخة Docker بسيطة لنشرها على خادمك،
- أو صفحة GitHub Actions لنشر تلقائي على GitHub Pages،
- أو إضافة فحص حالة السيرفرات الآلي مع جدول حالة.

أخبرني أي خيار تريده وسأجهّزه فوراً.
